helperFunctions.database module¶

class helperFunctions.database.ConnectTo(*args, **kwds)¶

Bases: typing.Generic

Open a database connection using the interface passed to the constructor. Intended to be used as a context manager.

Parameters

  • connected_interface – A database interface from the storage module (e.g. FrontEndDbInterface)

  • config – A FACT configuration.

Example
with ConnectTo(FrontEndDbInterface, self.config) as connection:
     query = connection.firmwares.find({})
helperFunctions.database.is_sanitized_entry(entry)¶

Check a database entry if it was sanitized (meaning the database entry was too large for the MongoDB database and was swapped to the file system).

Parameters

entry (Any) – A database entry.

Returns

True if the entry is sanitized and False otherwise.

Return type

bool
