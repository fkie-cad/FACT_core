helperFunctions.docker module¶

helperFunctions.docker.run_docker_container(image, timeout=300, command=None, reraise=False, mount=None, label='Docker', include_stderr=True)¶

Run a docker container and get its output.

Parameters

  • image (str) – the name of the docker image

  • timeout (int) – a timeout after which the execution is canceled

  • command (Optional[str]) – the command to run in the container (optional)

  • reraise (bool) – re-raise exceptions if they occur (timeout and docker exceptions)

  • mount (Optional[Tuple[str, str]]) – specifies a directory that gets mounted into the container; structure: (path_inside_container, source_path)

  • label (str) – label used for logging output

  • include_stderr (bool) – include stderr of the container in the output

Returns

the output of the docker container

Return type

str
